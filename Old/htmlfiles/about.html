About us

This dictionary puts a twist on your typical dictionary website. While providing the same concepts and core aspects of a regular dictionary, we include a vast amount of games to keep you entertained and knowledgeable. Our minimalistic website provides the most basic and simple presentation of a dictionary.

This website was created for an INST377 Dynamic Web Applications class at the University of Maryland. Contributors include: Abraham Lefevre, Afraah Goshu, Brandon Dang, Brian Lee, and Nikita Alexandrovich Lukyanov.

APIs Used

The APIs used and referenced to create this website are:
DictionaryAPI
Wiktionary (is the main reference for DictionaryAPI)
WordsAPI
RandomWordsAPI
